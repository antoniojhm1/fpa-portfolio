# FORGE — FP&A Analytics Portfolio
### Antonio Heredia · Economics & Finance · Toronto Metropolitan University

---

## About this project

I built this project to develop and demonstrate the technical skills 
required in modern FP&A roles  SQL, Python, Power BI, and data modeling  
while completing my co-op at State Street as a Pricing Analyst.

FORGE is a fictional B2B SaaS company I designed from scratch. 
All data is synthetic, generated by a Python script with realistic 
business assumptions across three customer segments: Enterprise, 
Mid-Market, and SMB.

The goal was to simulate a real FP&A workflow end to end from raw 
data to executive reporting and document every step so the thinking 
behind each decision is visible.

---

## What I built

**Data pipeline**
- Designed a relational database schema (SQLite) with 5 tables: 
  customers, contracts, revenue transactions, headcount, and budget
- Generated 500 customers and 7,375 revenue transactions using Python
- Wrote 7 core SQL queries for revenue, ARR, and variance analysis

**Python analysis (Jupyter notebooks)**
- Notebook 1: Revenue trend analysis and 6-month forward forecast
- Notebook 2: Cohort retention analysis with heatmap and segment curves
- Notebook 3: Budget vs actual variance engine with materiality flagging
- Notebook 4: Three-scenario analysis (upside / base / downside)

**Power BI dashboard**
- Connected directly to the SQLite database
- Built an executive dashboard with KPI cards, revenue trend, 
  segment breakdown, and budget vs actual by department
